Используйте CSS переменные уже сейчас

Плагин customProperties позволяет изпользовать переменные css в проекте, не думая о поддержке в IE

:root {
  --accentColor: #01a5a5;
}
.block {
  color: var(--accentColor)
}
.block {
  color: #01a5a5
}

Импортируйте стили в главный файл

Плагин importConcatinate позволяет разделять таблицу стилей на файлы и их импортировать в один файл

@import: url('variables.css')
.block {
  color: var(--accentColor)
}
:root {
  --accentColor: #01a5a5
}
.block {
  color: var(--accentColor)
}

Используйте функции для написания черного и серого цветов проще

Плагин grayFunction добавляет две функции gray() и black().
Функция gray() создает серый цвет из двух символов.
Функция black() позволяет использовать только прозрачнойсть для написания полупрозрачного черного цвета

.block {
  color: gray(80);
  background-color: black(0.5)
}
.block {
  color: #808080;
  background-color: rgba(0, 0, 0, 0.5)
}

Минифицируйте свой CSS-код

Плагин cssMinify уменьшает CSS-код, удаляя все переносы строк и лишние пробелы

.block {
  color: #808080;
}
.block {color: #808080}
