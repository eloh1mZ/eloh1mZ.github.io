Установка

Вы можете начать использувать PostCSSpy, выполнив несколько простых действий:

  • Установить Python 3.x(рекомендуестся 3.7)
    •
  • Скачать и разархивировать файлы в папку вашего проекта
  • Настроить пути в settings.py:
    9
    src = 'src/css/'
    10
    dist = 'dist/css/'
    11
    FileName = 'style.css'

Плагины

Установка плагинов заключается в:
  копировании файла с плагином в папку plugins;
  вставке текста с первой таблицы на 3 или более строку;
  вставке текста из второй таблицы в функцию Functions()

ВНИМАНИЕ Аргументы только первой функции должны быть (src, dist, FileName). Остальные - (dist, dist, FileName)

customProperties ⇩

>3
from plugins.customProperties import *
def Functions():
  customProperties(src, dist, FileName)
:root {
  --accentColor: #01a5a5;
}
.block {
  color: var(--accentColor)
}
.block {
  color: #01a5a5
}

>3
from plugins.importConcatinate import *

>3
from plugins.grayFunction import *
